## ⚙️ How It Works

This application is written in Python and uses the **Pillow (PIL)** image library to handle image processing and conversion.  
The image processing logic itself is provided by the library; this project focuses on the application logic and usability.

### Transparency Limitations
Some image formats do not support transparent backgrounds.

**Example:**
This behavior is a limitation of the image formats, not the application.  
If transparency is required, consider using an online background removal tool before converting.

### Image Quality
Low-resolution images may cause issues when converting to certain formats.  
For best results, always use high-quality images.

> ℹ️ If the output format is an icon, low resolution may be acceptable.
