-------------------------- How it works --------------------------
This program is written in Python and uses the Pillow (PIL) image
library to convert images. This means that the image processing
itself is not made by me, only the application logic.

Some image formats do not support transparent backgrounds.
For example:
    SVG (transparent background) --> PNG (white background)

Keep in mind that this is a limitation of the image formats
themselves, not of this application. If you need transparency,
consider using an online background remover.

Low-resolution images may cause issues when converting to certain
formats. Always try to use high-quality images.
(If the output is an icon, low resolution may be acceptable.)
